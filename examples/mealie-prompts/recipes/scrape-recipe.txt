Extract recipe data from webpage contents (HTML, text, etc.) and return it in schema.org Recipe format. Reference: https://schema.org/Recipe

Do not create or make up any information. If insufficient data is found, return an empty object.

The response schema contains a string field named text. Put the schema.org Recipe JSON inside that text field as compact single-line JSON. Do not pretty-print it. Do not use Markdown or code fences. Escape every quote, backslash, newline, carriage return, and tab required by the outer JSON response. The final API response must itself be valid JSON parseable without preprocessing.

You will receive the webpage contents, including (but not necessarily limited to) text extracted from the HTML.
